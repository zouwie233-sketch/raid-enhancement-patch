# Raid Enhancement Patch 0.8.9.7 安装说明

## 文件

- `raid_enhancement_patch-0.8.9.7-security-support-items.jar`

## 安装

把 JAR 放入 Minecraft 1.21.1 + NeoForge 21.1.234 的 `mods` 文件夹。

## 配置

启动一次游戏/服务器后会生成：

- `config/raid_enhancement_patch/village_security.properties`
- `config/raid_enhancement_patch/battle_support_items.properties`

本版本新增的战备令牌系统由 `battle_support_items.properties` 控制。

## 注意

- 只新增物品驱动的战备支援系统。
- 不修改 HUD。
- 不修改第 8 波桥接。
- 不修改第 9～11 波额外波状态机。
- 不修改 Raids Enhanced 特殊袭击者刷新逻辑。
- 不清除/传送袭击者。
- 不替代原版 Raid 胜负状态。
