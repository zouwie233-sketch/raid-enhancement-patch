# 0.8.9.8.7：顶部袭击标题稳定化与旧 HUD 清理

## 目标

- 取消旧的客户端自定义灰底 HUD。
- 不再使用下方 Actionbar 兜底显示。
- 波次信息只写入原版顶部袭击 BossBar 标题。
- 额外波标题显示为“最后攻势！”。
- 第 8 波不再被标为最后攻势；只有第 9～11 波属于最后攻势。

## 显示格式

- 普通波：`袭击 第 X / Y 波`
- 额外波：`袭击 第 9 / 11 波｜最后攻势！`
- 清剿中：`袭击 第 3 / 8 波｜清剿 02:15`

## 波次结构

继续保留 0.8.9.8.5/0.8.9.8.6 的 RaidWaveAuthority：

- 1～8 波由原版 Raid 负责。
- 9～11 波由模组额外波负责。
- 目标波次只认权威计划表，不再由 groupsSpawned 反推。

## 未改动

- 不改胜利奖励。
- 不改 JSON 奖池。
- 不改战场扫荡。
- 不改村庄恩情。
- 不改 Raids Enhanced 方块回滚和掉落物清理。
