# Rebuild Instructions

## Recommended command

Linux/macOS:

```bash
./gradlew build
```

Windows:

```bat
gradlew.bat build
```

## Expected output

```text
build/libs/raid_enhancement_patch-0.9.0.0-village-favor-v2-profession-gifts-alpha.jar
```

## Important

This project is for Minecraft `1.21.1`, NeoForge `21.1.234`, Java `21`, and Gradle plugin `net.neoforged.moddev 2.0.107`.

The Gradle Wrapper is configured for Gradle `8.12`. If the machine is offline, Gradle `8.12` and all required dependency caches must already exist locally.
