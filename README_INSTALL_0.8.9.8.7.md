# 安装说明：raid_enhancement_patch-0.8.9.8.7-bossbar-title-stability-hotfix

1. 删除 mods 文件夹内旧版本 `raid_enhancement_patch-*.jar`。
2. 只放入：`raid_enhancement_patch-0.8.9.8.7-bossbar-title-stability-hotfix.jar`。
3. 不要多个版本共存。
4. 建议删除运行期波次快照：`config/raid_enhancement_patch/raid_session_lifecycle.properties`。
5. 本版禁用旧自定义灰底 HUD，波次信息只并入原版顶部袭击 BossBar 标题。
