# raid_enhancement_patch 0.8.8.9-village-security-strict-rules

基线：0.8.8.8-village-security-preview。

本版本只加入两个严格规则层：

1. 村民袭击期间禁止外部回血 / 生命值钳制
   - 村庄安防系统接管村民生命压力后，首次纳入本场袭击的村民会记录 allowedHealth。
   - 袭击期间如果外部药水、光环、信标、祝福或其他模组把村民生命值抬高到 allowedHealth 以上，会压回 allowedHealth。
   - 安防铁傀儡全灭造成的 4 点内部惩罚伤害会更新 allowedHealth，因此不会被旧保护层或禁回血逻辑抵消。
   - 袭击胜利时解除限制并恢复存活村民满血；非胜利结束时解除限制但不额外恢复。

2. 袭击开始时清理多余普通未命名铁傀儡
   - 简单最多保留 3 个普通编外铁傀儡。
   - 普通最多保留 2 个普通编外铁傀儡。
   - 困难最多保留 1 个普通编外铁傀儡。
   - 不清理本补丁安防铁傀儡。
   - 尽量不清理命名铁傀儡。
   - 每场袭击只在开始时执行一次。

明确没有改动：HUD、第 8 波到第 9 波桥接、第 9～11 波额外波状态机、离场 / 回归冻结、Raids Enhanced 特殊袭击者生成、原版 Raid 失败状态、袭击者清理或传送。
