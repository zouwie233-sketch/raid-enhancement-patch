# 安装说明：0.8.9.6.1-security-buff-repair-hotfix

1. 删除旧版 `raid_enhancement_patch-0.8.9.6-village-security-deployment-doctrine.jar`。
2. 放入 `raid_enhancement_patch-0.8.9.6.1-security-buff-repair-hotfix.jar`。
3. 保留原配置文件 `config/raid_enhancement_patch/village_security.properties`。
4. 重启游戏或服务器。

建议测试：

- 第 1 波新刷安防铁傀儡是否出现抗性 1 级。
- 受伤安防铁傀儡进入下一波后是否恢复生命。
- 新刷和重新部署逻辑是否仍然正常。
