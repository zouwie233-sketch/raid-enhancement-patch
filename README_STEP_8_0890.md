# raid_enhancement_patch 0.8.9.0-village-security-messages-performance

基线：0.8.8.9-village-security-strict-rules。

本版本只做两个方向：

1. 村庄安防系统聊天栏信息反馈。
2. 村庄安防系统低风险性能整理。

## 新增信息反馈

提示前缀统一为：`[村民防卫同盟]`。

叙事设定：普通未命名铁傀儡被称为普通雇佣军，只负责日常防卫；袭击进入战役级状态后，村民防卫同盟派遣临时安防铁傀儡接管防线。

触发场景：

- 第 1 波初始入场提示。
- 后继波次需要补充安防铁傀儡时提示。
- 后继波次只需要修复安防铁傀儡时提示。
- 每波安防铁傀儡全灭、防守失败时提示。
- 袭击胜利时提示。
- 袭击失败 / 非胜利结束时提示。

每类场景内置多条随机中文文本，避免每次袭击都显示同一句。

## 性能整理

- 村民扫描改为 per-raid UUID 缓存。
- 村民缓存默认每 100 tick 刷新一次。
- 安防铁傀儡全灭检查仍保持 20 tick 级别，避免防守失败判定迟钝。
- 安防铁傀儡发光刷新默认降低到每 80 tick 一次，发光持续 160 tick，不影响视觉稳定。
- 普通铁傀儡清理仍然每场袭击只执行一次。
