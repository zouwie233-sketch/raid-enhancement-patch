# 安装说明：raid_enhancement_patch-0.8.9.8.2-hard-omen1-wave8-bridge-hotfix

## 基线

本版本基于 `0.8.9.8.1-world-tick-accessor-hotfix`。

## 修复目标

修复困难 + 不祥 1 级袭击中出现的波次不一致问题：

- HUD 显示 8 波。
- 实际原生袭击只生成到第 7 波。
- 旧逻辑误以为原版会自己生成第 8 波，因此没有补刷自定义第 8 波。

## 修复内容

1. 将困难 + 不祥 1 级的计划目标波数明确调整为 8 波。
2. 不再把理论上的“原版不祥额外波估算”当成已经生成的原生波次。
3. 只有 `groupsSpawned` 实际观测超过原生安全波数时，才承认原版已经真的生成更多波。
4. 如果原版在第 7 波后结束，则由自定义额外波桥接系统补出逻辑第 8 波。
5. 保留 `0.8.9.8.1` 的 LevelTickEvent 反射兼容热修。

## 安装

1. 删除 mods 文件夹内旧版本 `raid_enhancement_patch-*.jar`。
2. 只放入：`raid_enhancement_patch-0.8.9.8.2-hard-omen1-wave8-bridge-hotfix.jar`。
3. 不要多个版本共存。

## 测试重点

1. 困难 + 不祥 1：确认 HUD 显示 8 波，并且实际出现第 8 波。
2. 困难 + 不祥 2：确认仍为 8 波。
3. 困难 + 不祥 3/4/5：确认仍为 9/10/11 波，不提前结算。
4. 确认胜利结算、战场扫荡、村庄恩情仍正常。
