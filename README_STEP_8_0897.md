# 0.8.9.7-security-support-items 工作说明

基线：`0.8.9.6.2-villager-effect-visibility-hotfix`。

## 新增内容

本版本新增“村民防卫同盟战备令牌”系统，玩家可在村庄袭击期间消耗物品主动支援当前战场安防力量。

新增物品：

- 初级力量战备令牌：安防铁傀儡力量 I，5 分钟。
- 高级力量战备令牌：安防铁傀儡力量 II，5 分钟。
- 初级圣盾战备令牌：安防铁傀儡获得 20 点临时生命，5 分钟。
- 高级圣盾战备令牌：安防铁傀儡获得 50 点临时生命，5 分钟。
- 初级迅捷战备令牌：安防铁傀儡速度 I，5 分钟。
- 高级迅捷战备令牌：安防铁傀儡速度 II，5 分钟。
- 追猎战备令牌：安防铁傀儡速度 II + 力量 I，5 分钟。
- 初级炽火战备令牌：安防铁傀儡火焰抗性，5 分钟。
- 高级炽火战备令牌：安防铁傀儡火焰抗性 + 生命恢复 I，5 分钟。
- 雇佣兵战斗令牌：召唤 2 名临时雇佣兵铁傀儡，10 分钟。

## 雇佣兵规则

- 默认仅袭击期间可使用。
- 每名玩家默认最多同时拥有 2 名雇佣兵铁傀儡。
- 雇佣兵铁傀儡持续 10 分钟，到期撤离，不掉落物品。
- 契约期间免疫玩家伤害。
- 全程获得抗性 I、力量 I、速度 I。
- 会尝试跟随使用者，距离过远时传送至玩家附近。
